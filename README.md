# java-web-maven-spring-thyme-secure-single

## Description
A POC for springboot secure web app with thymeleaf support.

## Tech stack
- java
- maven
  - springboot
  - thymeleaf
  - bootstrap

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
Available at http://localhost
Login with id: user and password: pass

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
